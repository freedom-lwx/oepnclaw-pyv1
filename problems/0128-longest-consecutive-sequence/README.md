# 128. 最长连续序列

**难度:** Medium

**链接:** https://leetcode.com/problems/longest-consecutive-sequence/

## 题目描述

给定一个未排序的整数数组 `nums`，找出数字连续的最长序列的长度。要求算法的时间复杂度为 O(n)。

## 思路

**Set - 遍历元素起点找最长连续序列**

1. 将所有元素加入 Set
2. 遍历每个元素，如果 `num-1` 不在 Set 中，说明是序列起点
3. 从起点开始向下枚举，找最长连续序列

## 复杂度

| 复杂度 | 值 |
|--------|-----|
| 时间 | O(n) |
| 空间 | O(n) |

## 关键点

- 只从序列起点开始枚举，避免重复计算
- 用 Set 实现 O(1) 查找
